원격 서버 관리
-서버실에 서버 장비가 있고, 사무실에서 원격으로 접속해서 관리를 해야 한다.

1.Telnet:TCP 23 Port 를 사용함, TUI(명령어로 관리), Client와 서버의 데이터 패킷을 암호화 하지 않음
2.SSH: TCP 22 Port를 사용함, TUI(명령어로 관리),  Client와 서버의 데이터 패킷을 암호화 해서 사용.
     -리눅스 OS를 설치하면 SSH가 설치되어 있고 방화벽에서 기본적으로 오픈되어 있음
(SSH는 기본적으로 사용할줄 알아야 한다.)

3.VNC:원격의 리눅스에 X-Windows 가 설치되어 있으면 GUI환경에서 원격 관리가 가능
4.xrdp:VNC보다 향상된 기술, 원격의 GUI로 관리

5.rdp:TCP 3389 Port를 사용함, client와 server를 암호화 통신하면서 GUI 환경으로 관리(WINDOWS SERVER)/를 관리할때 쓴다.

SSH 설치:우분투 리눅스에서 , ubuntu 계정에서 ==>super user (root)/최고 관리자 계정으로 실행:sudo
$ sudo apt update                                           :apt update :인터넷 레파지토리에서 업데이트를 진행을 한다.
$ sudo apt install openssh-server  :SSH 서버를 설치

$ sudo systemctl status ssh    :SSH의 서비스 작동 유무를 확인
$ sudo systemctl enable ssh :  운영체제가 부팅이 될때 자동으로 서비스 시작
$ sudo systemctl start ssh :   SSH 서비스 시작
$ sudo systemctl stop ssh : SSH 서비스 중지
$ sudo systemctl restart ssh : SSH 서비스 다시 시작

$ sudo ufw allow ssh :SSH를 방화벽에서 오픈
